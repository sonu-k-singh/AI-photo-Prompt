# 🤖 AI Photo Prompt Website

This is a simple and user-friendly website where you can find ready-made AI photo prompts to generate amazing images using AI tools.

## 🚀 Features

* 📋 One-click copy prompt button
* 🎨 Ready-to-use AI photo prompts
* ⚡ Fast and lightweight design
* 💻 Built using HTML, CSS, and JavaScript

## 🧠 How It Works

1. Browse different AI prompts on the website
2. Click the **Copy** button
3. Paste the prompt into any AI image generator (ChatGPT, Gemini, etc.)
4. Generate your AI image instantly

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript

👉 https://sonu-k-singh.github.io/AI-photo-Prompt/

This project is made to help users easily access high-quality prompts without needing to create them manually.
